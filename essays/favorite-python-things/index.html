My Favorite Things About Python

A Running list of the things that I enjoy most about python and why I think they make the language elegant and powerful.

Python is certainly one of my favorite programming languages and it's just because it's so fun to write. One of the other things I love about python is how frequently I learn new things about it that I never would have expected to be in a programming language. What you'll find here is a list of how some of my favorite features of python work and why I think they're so cool.

String and List Indexing

This is a super basic feature but I think it expertly demonstrates how intuitive and clever python can be. This isn't your grandma's substring() method.

>>> list = [0,1,2,3,4,5]
>>> list[:2]
[0, 1]

# If we want to reverse a list we can just do.
>>> list[::-1]  # This means, take the whole list (list[::]) but step by -1.
[5, 4, 3, 2, 1, 0]

# We can do this with strings too! We get all the fun of being able to work with a char*
# with all of the cleverness of a high-level language like python!
>>> string = "Hello World!"
>>> string[:3]  # take from the beginning of the string to index 3 (exclusive)
'Hel'
>>> string[3:]  # take from index 3 (inclusive) to the end of the string.
'lo World!'

*args, **kwargs, and Optional Parameters

Normally we know our parameters and can define our functions simply like:

def product(a, b, c):
    return a * b * c

However this can be a bit limiting because what if we want to be able to get the product of an arbitrary number of integers? This is where *args comes in handy. We can use an unlimited amount of positional arguments in *args and get what we want.

def product(*args):
    product = 1
    for i in args:
        product *= i
    return product

We can call this function a multiple of ways:

>>> product(1,2,3,4,5)
120
>>> a = [5,10,2]
>>> product(*a)
100

We can also use a variable number of keyword parameters if we use **kwargs.

def print_kwargs(**kwargs):
    for key, value in kwargs.items():
        print key, value

We can also pass in whole dictionaries as arguments to our function!

def print_person(name, age, occupation):
    print name, age, occupation

This will work if we call the function with a dictionary that has the keys name, age, and occupation.

>>> john = {'name': 'John', 'age':30, 'occupation':'fireman'}
>>> print_person(john)
'John 30 fireman'
